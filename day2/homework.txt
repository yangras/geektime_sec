四、分别使用行内式、内嵌式、引入外部文件的方法造成网页弹窗，要求触发弹窗的 JavaScript 命令不止一种（alert、print、prompt）。
行内
<button onclick="alert('Hello')">点击弹窗</button>
内嵌式
<script>
    function showPopup() {
        alert('Hello');
    }
</script>

<button onclick="showPopup()">点击弹窗</button>

引入外部文件
<script src="popup.js"></script>
在popup.js中如下内容
<button onclick="showPopup()">点击弹窗</button>
